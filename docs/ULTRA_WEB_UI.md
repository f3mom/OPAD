# OPAD Ultra Web UI

OPAD Ultra exposes every major Attack-Defense workflow through the browser. Start at `/ultra` after setup. Each module has a page, API endpoints, safe dry-run actions, integration notes and artifacts.

Modules include Game, Scope, Targets, Services, Flags, Submitter, Exploits, Agent, Patching, Checker Replay, Traffic, Capture, Filters, Monitoring, Automation, Notifications, RBAC, Secrets, Incidents, Backups, DevOps, Plugins, Lab and Reports.

Risky actions remain gated: scope validation, checker replay, healthcheck and rollback plan are required before apply/deploy flows.
