# First 10 minutes

1. Confirm the CTF scope, own team ID and allowed CIDRs.
2. Generate target list and verify own team is excluded.
3. Register services and run healthchecks.
4. Create snapshots before editing anything.
5. Start capture on the game interface with management ports excluded.
6. Sync services and flag patterns to Packmate or another traffic provider.
7. Run checker-like tests and save good traffic for replay.
8. Test submitter in dry-run or organizer sample mode.
9. Run only harmless connectivity probes until exploits are reviewed.
